Getting Started

Master the basics and set up your first project

1

Create Your First Project

  • Click "Create New Project" on the dashboard
  • Enter a project name and description
  • Select the project type (Education, Research, Business, etc.)
  • Add your name as the creator for attribution
  • Click "Create Project"
2

Organize with Cohorts

  • Open your project and click "Create New Cohort"
  • Name your cohort (e.g., "Q1 Data", "Student Records")
  • Add a description explaining the cohort's purpose
  • Include your name as the creator
3

Upload Your Data

  • Navigate to a cohort and click "Upload Spreadsheet"
  • Select your Excel file (.xlsx or .xls)
  • Provide a name and creator information
  • The system will automatically analyze your data

Spreadsheets

Powerful Excel file management and analysis

Uploading Spreadsheets

  • Supported Formats: .xlsx and .xls files
  • Automatic Analysis: System detects columns, data types, and structure
  • Smart Headers: Automatically identifies column headers

Online Spreadsheet Editor

Professional Excel-like editing with advanced features

Excel-like Interface

  • Cell Navigation: Use arrow keys, Tab, and Enter to move between cells
  • Formula Support: Use =SUM(), =AVERAGE(), =COUNT(), etc.
  • Auto-save: Changes are automatically saved
  • Real-time Updates: See formula results immediately

Formatting Tools

  • Text Formatting: Bold, italic, underline, strikethrough
  • Alignment: Left, center, right alignment
  • Number Formatting: Currency, percentage, date, number formats
  • Cell Highlighting: Color-code cells for better organization

Advanced Functions

  • Math Functions: SUM, AVERAGE, COUNT, MAX, MIN
  • Text Functions: CONCATENATE, LEN, UPPER, LOWER
  • Cell References: Use A1, B2, etc. in formulas
  • Range Support: Use A1:B10 for ranges
